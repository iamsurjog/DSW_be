# DBMS Project Backend
## Installation
To download the repo run
```bash
 git clone https://github.com/iamsurjog/dsw_be.git
 py -m venv ./.venv
 pip install -r requirements.txt
```

## Running it
### Windows
```bash
.\.venv\Scripts\activate
fastapi run main.py
```

### Linux
```bash
source .venv/bin/activate
fastapi run main.py
```
